# Pendulum Lab

Mechanics lab for a simple pendulum, comparing the nonlinear equation against
the small-angle approximation.

## Run

```sh
make console
./build/pendulum_console
```

## Plot

```sh
make plots
```

Outputs:

```text
build/pendulum_nonlinear.csv
build/pendulum_small_angle.csv
build/pendulum_angle.svg
build/pendulum_energy.svg
```
